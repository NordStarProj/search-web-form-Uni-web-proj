## Инструкция по локальному запуску

1. Открываем папку проекта в VSC / Любом другом удобном вам редакторе
2. В терминале прописываем `yarn`
3. После иницализации yarn вписываем `yarn run dev`
4. Откройте ссылку из терминала.
